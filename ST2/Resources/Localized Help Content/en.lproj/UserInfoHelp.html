Pull-To-Refresh
Swipe downward in a scrolling motion to activate the Pull-To-Refresh feature. This causes information for the current user to be fetched from the Silent Circle servers, for example, to fetch the user's photo. (See User Photo for more)

Note that fetching information from servers may take more or less time, depending on network conditions.

User Photo
The user photo may come from several different sources, by priority:

  1. Added by you in the Edit view.
    Tap Edit button at top right of the contact info view, then tap the photo in the Edit view. You will be presented with options to take a photo (if device camera is available), or choose photo from Photos library. This is the top priority photo.
    Note that you may also delete a photo, in which case, the photo displayed will fall back to a lower priority photo if available.
  2. The photo linked to this Silent Contact from your system Contacts.
    If the current user photo has been linked to a Silent Contact user from a system Contacts user, this will be the preferred photo until you add a new as described before.
  3. A photo the user has uploaded to their Silent Circle account settings web page.
    If the current user has uploaded a photo, Pull-To-Refresh will fetch their photo from the web. This photo is the "last resort" lowest priority photo to be displayed.
    Note that if you add a photo for this user and then delete it, the web photo will be displayed if available.

    Phone Icon
    Tap the phone icon to switch to the Silent Phone app, if installed on this device, ready to place a call to this user.

    Message Icon
    The action resulting from a tap on the Message Icon depends on where you are in the app.

    In the Messages view of a conversation, you can tap on the conversation title for the Conversation Details view. Then a tap in the area of the user name/photo will present the User Info view. Also in the Messages view of a conversation, a tap on the security button at top right will present the Conversation Security view which also navigates to the User Info view in the same way.
    In either case, tapping on the Message Icon will simply return you back to the Messages view from where you started.

    If you arrive at the User Info view by tapping a contact in the Silent Contacts view, a tap on the Message Icon will attempt to pop you into the Messages view of a conversation with that user if there is an existing conversation, or will start a new conversation with the selected user.

    Notes
    The Notes section contains notes you have previously entered for this contact. To enter or modify the contact notes, enter the Edit view by tapping on the Edit button at top right of the User Info view.
