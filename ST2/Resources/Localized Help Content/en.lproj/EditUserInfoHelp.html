User Photo
When you tap the photo you will be presented with options to update the photo for this contact:

  • Delete Photo
    If you have previously added a photo in this app for this contact, the option to delete the photo will be presented.
  • Take Photo
    If the camera on this device is available, the option to take a photo with the camera will be presented.
  • Choose Photo
    Selecting this option will present the Photos library from which you can choose a photo.

Link with Contacts Book
When tapped, your Contacts app list will be presented. Select a contact to import their information to the user being edited. The selected contact's first name, last name, organization, photo, and Notes, will be applied.

Tap the Save button to save changes, or tap the Cancel button to go back to the user info display view without saving the changes.

Note that linked contact information may be further modified while in the Edit view. For example, enter a space in the last name field to remove the linked last name info.

Also note that the contact's original information is not modified in the Contacts app when edits are applied to this Silent Circle contact.

Once the current user is linked to the Contacts contact, the title will change to "Unlink from Address Book". Then the tap action will unlink the current user from the previously linked contact.

Create Public Key
If you have reason to believe your existing public key may be compromised, or if your company requires you to change your public encryption key at regular intervals, you can use this feature to update your public key. When you tap Create Public Key a confirmation message will be presented. When confirmed, an new public encryption key will be created and stored on the Silent Circle servers. There may be some delay with this process depending on network conditions.

Delete Contact
Tap to delete this contact from Silent Contacts and completely remove all conversations with this user.

This option will not be displayed if viewing your own user information.

Note that if the contact being deleted is linked to a Contacts user, the Contacts user information will not be modified or deleted.
